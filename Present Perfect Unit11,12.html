Unit 11: How long have you (been) ... ?

핵심 개념

present perfect는 과거에 시작해서 지금까지 계속되는 상태/행동을 말할 때 사용한다.

중요한 포인트

  • 현재 상태를 단순 현재로 말할 수는 있음
    ex) They are married.
  • 하지만 기간을 물을 때는
    How long have they been married?
  • 기간을 포함해 말할 때도
    They have been married for 20 years.
  • not: They are married for 20 years.

예시 (Examples)

He's been in the hospital since Monday.

We've known each other for a long time.

How long have they had their car?

She hasn't been waiting very long.

Present Perfect: Simple vs Continuous

1) Continuous가 자주 쓰이는 경우

특히 How long과 함께 자주 사용

  • I've been learning English since January.
  • It's been raining all morning.
  • Richard has been doing the same job for 20 years.
  • How long have you been driving?

2) Continuous를 잘 안 쓰는 동사

상태동사: know, like, have 등

  • How long have you known Jane?
  • I've had these shoes for ages.

3) 둘 다 가능한 경우

live, work

  • Julia has been living / has lived here for a long time.
  • How long have you been working / worked here?

4) 단순형만 쓰는 경우

always와 함께는 보통 simple

  • I've always lived in the country.

Unit 12: for and since

for

기간 (duration)

Structure: for + 기간
• for two hours
• for six months
• for a long time

Examples:

  • We've been waiting for two hours.
  • Emily has been working here for six months.

since

시작 시점 (starting point)

Structure: since + 시점
• since 8:00
• since Monday
• since April / 2009

Examples:

  • We've been waiting since 8:00.
  • Emily has been working here since April.
  • I haven't seen Tom since Monday.

When ... ? vs How long ... ?

When ... ?

언제 시작/발생했는지 묻는다

→ 보통 past simple
Q When did it start raining?
A It started raining an hour ago / at 1:00.

How long ... ?

얼마 동안 지속되었는지 묻는다

→ 보통 present perfect
Q How long has it been raining?
A It's been raining for an hour / since 1:00.

또 다른 예시 비교

When did Joe and Kate first meet?
→ They first met when they were in high school.

How long have they known each other?
→ They've known each other for a long time / since high school.

Core Rule Comparison

기능 질문/표현 형태 시제 예문
현재까지의 지속 기간 묻기 How long have you been ... ? Present Perfect How long have you been studying English?
기간 표현 for + 기간 Present Perfect I've studied English for two years.
시작 시점 표현 since + 시점 Present Perfect I've studied English since 2024.
시작한 시점 묻기 When did ... start / happen? Past Simple When did you start studying English?
💡 When은 시작 시점, How long은 지속 기간에 초점을 둔다.

Creative Extension: AI Grammar Quiz Game

기획 의도

교재 문법을 단순 설명에 그치지 않고, 학습자가 직접 문제를 풀며 개념 차이를 체감하도록 확장했습니다.

영어문법1 강의를 수강하는 학생들이 시험 직전에도 핸드폰으로 간단하게 퀴즈를 풀면서 교재 예문을 자연스럽게 외울 수 있도록 하는 것이 목표입니다. (책을 펴서 보는 비효율을 개선)

게임 핵심 기능

  • 난이도 선택: 하 / 중 / 상
  • 교재 기반 예문 약 30개 탑재
  • 리롤 버튼 클릭 시 랜덤 10문제 출제
  • 객관식 또는 빈칸형 문제 제공
  • 제출 후 점수와 학점 출력
Result System

결과 체계 예시

10개 정답 A+
9개 정답 A
8개 정답 B+
7개 정답 B
6개 정답 C
5개 이하 Review Needed

Why This Is Creative

1. 정적 요약 → 동적 퀴즈

교재 문장을 눈으로만 읽는 정적 요약에서 벗어나, 퀴즈 형태의 동적 경험으로 변형했습니다.

2. 즉각적인 피드백 사이클

문법 설명 → 즉시 적용 → 결과 피드백의 완벽한 구조를 HTML/JS로 구현했습니다.

3. 학습자 맞춤형 난이도

난이도별 구성을 통해, 자신의 학습 수준에 맞춰 점진적으로 실력을 키울 수 있습니다.

4. 미세한 문법 차이 구별

단순 번역이 아닌 for/since, when/how long, simple/continuous의 차이를 구별하게 설계했습니다.

검증 기준

1 교재와 일치하는가?
2 상태동사 예외가 정확히 설명되었는가?
3 When과 How long의 시제 차이가 명확한가?

5 AI가 만든 문제에 문법적 오류가 없는가?

비판적으로 본 부분 (AI의 한계 보완)

  • AI는 sometimes present perfect simple과 continuous의 차이를 지나치게 단순화할 수 있음
  • know, like, have 같은 상태동사는 continuous로 잘 쓰지 않는다는 예외를 반드시 확인해야 함
  • "How long"이 나온다고 무조건 continuous를 쓰는 것은 아님
  • "always"와 함께는 simple 형태가 자연스럽다는 점도 검토 필요

Conclusion

핵심 정리 (Summary)

  • How long은 현재까지의 지속 기간을 묻는다.
  • for는 기간, since는 시작 시점이다.
  • When은 시작 시점을 묻고, How long은 지속 시간을 묻는다.
  • Present perfect simple과 continuous는 문맥에 따라 구별해야 한다.
  • AI를 활용하면 문법 학습을 더 상호작용적으로 확장할 수 있다.
